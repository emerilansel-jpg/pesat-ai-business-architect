# Pesat AI Business Architect

AI-powered business consultant chatbot that helps businesses find the most valuable AI solutions.

## URL
https://apps.pesat.ai/advisor/

## Features
- Inline AI-generated visualizations within responses
- Multiple choice buttons for user interaction
- 6-step conversation flow (Research > Interview > Report > CTA)
- Admin panel with step prompt editing and testing
- Deep search integration (Tavily)
- Model selection (OpenAI GPT-4o/4.1/4.5/5, DeepSeek)

## Tech Stack
React 19 + TypeScript + Vite + Tailwind CSS + Framer Motion

## Deployment
Static build deployed to VPS at `/var/www/advisor/`
